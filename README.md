# OCDダンジョン (OCD Dungeon)

マインドフルネスの力で強迫観念（OCD）という名のモンスターを鎮める、癒やしと冒険の1.5人称ブラウザRPG。

![OCD Dungeon Conceptual Image](https://images.unsplash.com/photo-1518531933037-91b2f5f229cc?auto=format&fit=crop&q=80&w=1000)
> [!NOTE]
> このイメージはプロジェクトの温かみのある世界観を表現しています。

## 🌟 プロジェクト概要

「OCDダンジョン」は、強迫症（OCD）の症状やそれに伴う不安を、ファンタジー世界の「モンスター」として表現したマインドフルネス・サポートゲームです。プレイヤーは呼吸や感覚への集中を通じて、モンスター（強迫観念）を鎮め、自分自身の心に静寂を取り戻すことを目指します。

温かみのある**水彩画風のデザイン**と**手描き風のUI**が、安心できるプレイ空間を提供します。

## ✨ 特徴

- **ユニークなバトル「静寂の試練」**: タイマーが減る間、マインドフルネスな呼吸やグラウンディングを行い、心の「チャージバー」を満たすことで勝利します。
- **個性豊かなモンスター**: 
  - `ジロリン`（確認強迫）
  - `ドロル`（汚染強迫）
  - `チグハグ`（完璧/儀式）
  - `ボソリ`（加害恐怖/不道徳）
- **自分だけの挑戦**: 「カスタムモンスター」機能で、自分の気になる悩みや観念に名前をつけて対峙できます。
- **癒やしの報酬**: レベルアップするごとに、部屋を彩る「リラックスグッズ」（アロマキャンドル、ふかふかクッション、観葉植物など）がアンロックされます。
- **メダルコレクション**: 勝利数や難易度に応じたメダルを集めることができます。

## 🛠 技術スタック

- **Core**: TypeScript
- **Styling**: Vanilla CSS (水彩風・紙質テクスチャ)
- **Framework**: Vite
- **Assets**: Inline SVG (アイコン・モンスターデザイン)

## 🚀 セットアップ

### 必要要件
- Node.js (v18以上推奨)
- npm

### インストール
```bash
# リポジトリをクローン
git clone https://github.com/ryousuke971003/OCD-Dungeon-2.git

# ディレクトリに移動
cd OCD-Dungeon-2

# 依存関係をインストール
npm install
```

### 開発用サーバーの起動
```bash
npm run dev
```
起動後、ブラウザで `http://localhost:5173` にアクセスしてください。

## 🎨 デザインコンセプト

このゲームは「安心感」と「不完全さの受容」をテーマにしています。
- **カラーパレット**: 優しいパステルカラーと、目に優しいアースカラー。
- **テクスチャ**: 紙のような質感や木目調のボタン。
- **アニメーション**: ゆらゆらとしたSVGの動き。
